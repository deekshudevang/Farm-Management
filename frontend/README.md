# 💎 AgriSmart — Premium Interface

The high-performance, glassmorphic frontend for AgriSmart.

## 🚀 Execution
1. `npm install`
2. `npm run dev`

## 🎨 Design System
- **Framework**: React + Vite
- **Styling**: TailwindCSS + Custom Glassmorphic Layers
- **Icons**: Lucide React
- **Charts**: Recharts

## 📁 Key Directories
- `src/pages`: 10/10 Enterprise pages.
- `src/components`: Glassmorphic UI components.
- `src/services`: API bridge.
